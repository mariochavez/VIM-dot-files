Install
========

    cd ~
    git clone git@github.com:mariochavez/vim-dot-files.git ~/.vim
    ln -s ~/.vim/vimrc ~/.vimrc
    ln -s ~/.vim/gvimrc ~/.gvimrc
    cd ~/.vim
    git submodule init
    git submodule update

Update
========

    git submodule foreach git pull origin master
